# cpDesign
R codes for using conditional power to create optimal futility stopping boundareis for phase III trials with delayed treatment effects

cpDesign_fun.R contains all the lower level functions for the packages.

cpDesign_application.R provides examples.

Fig1.R is used for creating Figure1 in the manuscript.

FigS1.R is used for creating Figure1 in the supplementary.

get_Tables.R is used for creating tables.
